# 數字解碼研習社 - 網站專案

這是一個結合體重管理與生日數字解碼的身心健康導師網站。

## 專案結構
- `index.html`: 首頁
- `pain-points.html`: 破解痛點頁面
- `solution.html`: 解決方案頁面
- `core-value.html`: 核心價值頁面
- `contact.html`: 預約諮詢頁面
- `style.css`: 網站樣式表
- `script.js`: 互動腳本
- `assets/`: 網站圖片資源 (WebP 格式)

## 如何預覽
1. 下載此儲存庫。
2. 直接點擊 `index.html` 即可在瀏覽器開啟。
