# 版本更新紀錄 (v5_Mobile_RWD)

## 變更概述
在本次版本中，我們針對 v4 開發的「卡片問答遊戲」加入了深度的 RWD (Responsive Web Design，響應式網頁設計) 優化。目標是確保測驗在各種尺寸的行動裝置 (手機、平板) 上都能夠正常顯示、無溢位且提供最順暢的「拇指操作體驗 (Fat-finger friendly)」。

## 具體變更項目 (quiz.css)
1. **新增平板斷點 (`@media max-width: 768px`)**：
   - 卡片邊距縮小 (`padding: 40px 25px;`)，增加可用空間。
   - 解決網頁鍵盤彈出問題：將 `quiz-main-layout` 設定為 `align-items: flex-start`，配以 `padding-top: 80px`，這能防止手機螢幕上的軟體鍵盤遮擋住表單輸入框。
2. **拇指點擊友善化 (Touch Friendly)**：
   - 雙欄的 `.input-grid` 強制轉為單欄式堆疊。
   - 性別選擇卡片與所有 `.btn-primary`、`.btn-text` 按鈕改為 100% 寬度且加大內距，讓手指更容易準確點擊。
3. **字型動態縮小 (Adaptive Typography)**：
   - 電腦版碩大的標題和小字，在手機版有經過等比例微縮處理 (`h2` 下降至 1.4rem，字距行高優化)，確保即使題目文字很長也不會跌出卡片之外。
4. **新增超小螢幕特化 (`@media max-width: 380px`)**：
   - 特別為 iPhone SE 這類超級狹長的舊式螢幕，額外給了一層破版防護網 (縮更小的邊界與更充裕的高度)。

## 老師的設計巧思
- 行動版網頁最忌諱「字太小點不到」或是「選項卡在螢幕右邊滑不過去」。老師把所有的選項排版都從打橫 (Flex-row) 變成垂直堆疊 (Flex-column)，並且讓按鈕面積放到最大。
- 這樣客戶在等公車或躺在床上滑手機時，也能用單手拇指非常輕鬆地「點完所有題目送出」，這對你收集名單絕對是大大的加分！
